# 移除链表元素

题目来源：[203. 移除链表元素](https://leetcode.cn/problems/remove-linked-list-elements/description/)

给你一个链表的头节点 `head` 和一个整数 `val`，请你删除链表中所有满足 `Node.val == val` 的节点，并返回新的头节点。

## 示例

示例 1：

![](./removelinked-list.jpg)

```plain
输入：head = [1,2,6,3,4,5,6], val = 6
输出：[1,2,3,4,5]
```

示例 2：

```plain
输入：head = [], val = 1
输出：[]
```

示例 3：

```plain
输入：head = [7,7,7,7], val = 7
输出：[]
```

## 约束

- 列表中的节点数目在范围 `[0, 10^4]` 内
- `1 <= Node.val <= 50`
- `0 <= val <= 50`

## 解法速览

| 方法 | 核心思路 | 复杂度 |
| --- | --- | --- |
| 虚拟头节点 | 使用哨兵节点统一处理，遍历时删除所有匹配节点 | `O(n)` / `O(1)` |
| 递归 | 递归处理每个节点，返回处理后的链表 | `O(n)` / `O(n)` |
| 迭代（先处理头节点） | 先删除所有连续的目标头节点，再遍历删除中间节点 | `O(n)` / `O(1)` |
