# LCR 136. 删除链表的节点

题目来源：[LCR 136. 删除链表的节点](https://leetcode.cn/problems/shan-chu-lian-biao-de-jie-dian-lcof/)，即原「剑指 Offer 18」的新版编号。题意、数据范围一致，只是 LeetCode CN 在 2023 年起把大部分 Offer 题重新归到 LCR（LeetCode Coding Room）系列。

给定单向链表头指针 `head` 以及需要删除的节点值 `val`，删除该节点并返回新的头结点。题目保证链表节点值互不相同。

## 示例

```
输入：head = [4,5,1,9], val = 5
输出：[4,1,9]
```

```
输入：head = [4,5,1,9], val = 1
输出：[4,5,9]
```

## 约束

- 链表长度范围 `[0, 10^4]`
- 节点值均为整数，且互不相同

## 解法速览

| 方法 | 核心思路 | 复杂度 |
| --- | --- | --- |
| 虚拟头节点 | 哨兵节点统一处理头节点删除逻辑，遍历找到目标后跳过 | `O(n)` / `O(1)` |
| 双指针 | 使用前驱 + 当前指针，提前判断头节点是否需要删除 | `O(n)` / `O(1)` |

更进阶的写法还可以使用递归或单指针自适应版本，思路与此类似。
